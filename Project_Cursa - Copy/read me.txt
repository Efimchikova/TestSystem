   Универсальная тестирующая система

Эта программа позволяет вас создать универсальный тест

При создании файлов МОЖНО создавать все возможные названия!

Для создания не обходимо :
 1) Создать файл с вопросами
	Структура файла :
		а)Первая СТРОКА вопрос;
		б)Варианты ответов(максимум размер экрана/20);
		в)Написать разделительный знак "&" (без ковычек)
		г)перейти к подпункту а) или пункту д)
 		д)сохранить файл
 2)Создать файл с переменными
	A)Обычный тест(Где считаються кол-во правильных и в конце выводит кол-во отвеченных верно
			из всех вопросов:
	Структура файла:
		a) Первая строка Название переменной "True" (без ковычек)
		б) Вторая строка Написать "Set user clirence yoga"
		в) Третяя Строка "&"
	Б)"Как проф опросник" тест(Где выводиться к какому из переменных мы больше всего подходим,
		т.е. создав 20 переменных и пройдя тест нас определят к какому типу переменной мы подходим)
		а)Название переменной
		б)Значение переменной(выведет в конце)
		в)"&"
		г)перейти к подпункту а) или пункту д)
 		д)сохранить файл
 3)Создать файл с ответами

 		а)Если это 2.А тогда пишем test
		б)Если это 2.Б тогда пишем max
		в)Если тест типа профопросник пишем какую переменную мы хотим инкрементировать,если не хотим
		инкрементировать ставим знак "&"
		г)Если тест типа "Обычный" какой из ответов правильный пишем "True", иначе "&"
		д)В конце вопроса пишем "~"
		e)Переходим к п. в) или п.я)
		я)сохранить файл
 4)Заходим в пограмму и жмем кнопку decode
 5)Выбираем файл в ответами,и сохраняем закодираванный файл
 6)Заходим в файл data.bin (корневая папка) и добавляем строки
		а)Название теста
		б)Путь к файлу с вопросами
		в)путь к файлу с результетами
		г)путь к закодированному файлу
